HSBC业务线人脸识别能力解决方案

Project Description

应用场景：

HSBC员工之间要组织一个活动，但是由于员工数量众多，大多数员工之间并不认识，希望能够促进员工间的交流。

解决方案：

设计开发了遇见APP产品，员工只要安装在手机端且注册后。在员工活动时，对于不认识的人，只要需要用摄像头扫一扫，即可识别对方信息（包括姓名、爱好等等），方便员工之间相互了解交流。
